# Hệ thống quản lý thiết bị trường học

Đây là đồ án tốt nghiệp ngành Công Nghệ Thông Tin.

## Chức năng

- Quản lý thiết bị
- Ghi nhận bảo trì
- Theo dõi tình trạng thiết bị
- Quản lý người dùng

## Công nghệ sử dụng

- PHP
- MySQL
- HTML CSS
- JavaScript

## Cài đặt

1. Import file database trong thư mục database
2. Copy source code vào thư mục htdocs của XAMPP
3. Mở trình duyệt và chạy localhost
